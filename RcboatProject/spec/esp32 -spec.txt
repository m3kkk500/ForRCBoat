Features:
1. The module is an ESP-WROOM-32 development board, and ESP-WROOM-32 parameter specifications refer to the parameter specifications of the module.
2.802.11B/G/WIFI + BT module; Low power dual-core 32-bit processor can be used as an application processor;
3. Main frequency can reach up to 240MHz and computing power can reach up to 600dmips; 520KB SRAM built-in;
4. Support interfaces such as UART/SPI/I2C/PWM/DAC/ADC; Support OPENOCD debug interface;
5. Supports multiple sleep modes with a current as low as 6.5UA during deep sleep; Embedded LWIP and FREERTOS;
6. Support STA/AP/STA+AP mode; Support smart configuration/AIRKISS one-click deployment network;
7. Universal AT command, easy to use; Support local serial port upgrade and remote upgrade.



Description:
The main body of the module is TYPE-C module, which uses a USB to CP2102 serial port chip to expand the ESP-WROOM-32 interface. It can be directly connected to the computer for debugging, and data transmission is fast and stable. There is a power indicator light (red light) and working status indicator light (blue light) on it, which allows you to know the status of the module at any time. There are two touch buttons, one resets and the other allows the module to enter download program mode. Moreover, the pins on both sides of the development board can make it easier for developers to connect and use.
The module has powerful Bluetooth and WIFI functions. As a Bluetooth and WIFI development board, it can access rich resources. In addition, the module supports multiple development environments, expands its existing resources and greatly improves the ease of learning and development.
